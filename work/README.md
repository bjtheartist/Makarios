## Project Summary

**MAKARIOS** is a Christian discipleship and spiritual growth mobile web application.

### Core Features
- Daily verse
- Video courses (3)
- Guided prayers (20 across 6 categories)
- Bible trivia (6 books)
- Bible lessons with quizzes (3)
- User testimonies (5)
- Subscription paywall

### Tech Stack
- React 19 + TypeScript
- Vite build tool
- TailwindCSS (CDN)
- Lucide icons

### Current State
- Frontend: Complete
- Backend: Not started
- Auth: Simulated
- Payments: UI only

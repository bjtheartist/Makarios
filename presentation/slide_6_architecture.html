The Architecture

A component-based architecture makes the app modular, scalable, and easy to maintain

Single-Page Application

The app loads once and navigates between views without full page reloads, creating a fast, seamless, and app-like user experience.

Component-Based

Each feature (Home, Prayers, Trivia) is its own self-contained component, making the codebase modular and easy to extend.

Centralized State

The main App.tsx file manages the global state (user, current view, loading status), ensuring a single source of truth.

Static Data Layer

All content is currently stored in constants.ts for rapid prototyping, ready to be migrated to a scalable database.
